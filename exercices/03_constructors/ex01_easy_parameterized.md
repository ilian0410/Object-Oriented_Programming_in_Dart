# Constructors тАФ Easy

## Instructions

Create a `Laptop` class with:

- **Private fields**: `_brand` (String), `_ram` (int), `_price` (double)
- **Constructor**: parameterized constructor using shorthand `this.` syntax
- **Getters**: for all three fields
- **Method**: `specs()` that returns a string like: `"Brand: Dell, RAM: 16GB, Price: \$1200.00"`

In `main()`, create two `Laptop` instances and print their specs.

## Expected Output

```
Brand: Dell, RAM: 16GB, Price: $1200.00
Brand: Apple, RAM: 8GB, Price: $999.99
```

## Restrictions

- Use only one constructor (parameterized)
- All fields must be final
