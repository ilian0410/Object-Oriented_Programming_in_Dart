# Classes and Objects — Easy

## Instructions

Create a `Book` class with the following:

- **Fields**: `title` (String), `author` (String), `price` (double)
- **Constructor**: parameterized constructor using shorthand `this.` syntax
- **Method**: `displayInfo()` that prints the book details in the format: `"Title by Author — \$price"`

Then, in `main()`, create two `Book` objects and call `displayInfo()` on both.

## Expected Output

```
The Hobbit by J.R.R. Tolkien — $15.99
1984 by George Orwell — $12.50
```

## Restrictions

- Do not use named parameters
- All fields must be set via the constructor
