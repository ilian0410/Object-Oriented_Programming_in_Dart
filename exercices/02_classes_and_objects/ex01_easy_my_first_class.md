# Classes and Objects — Easy

## Instructions

Create a `Dog` class with:

- **Fields**: `name` (String, default `''`), `age` (int, default `0`), `breed` (String, default `''`)
- **Method**: `bark()` that prints `"Woof! I'm name"`
- **Method**: `describe()` that prints `"name is a age-year-old breed"`

In `main()`, create one `Dog` object, set its fields, and call both methods.

```
Woof! I'm Rex
Rex is a 3-year-old German Shepherd
```
